# README - Projet07-AJ

## Lancer le jeu

1. Installer Java (JDK 17 ou supérieur).
2. Ouvrir un terminal dans le dossier LAST BATTLE–Calculator-v1.
3. Exécuter la commande suivante :
   
   ./gradlew run

## Contenu de l'archive
- Code source : tous les fichiers `.java` (y compris les sous-dossiers `engine` et `patterns`).
- Fichiers de configuration Gradle : `build.gradle.kts`, `settings.gradle.kts`, etc.
- Rapport :
  rapport-page-de-garde.pdf
  rapport-structure-code.pdf
  rapport-design-patterns.pdf
